# Notification Sounds

This directory contains audio files for notification sounds.

## Required Files

- `notification.mp3` - The default notification sound file

## Adding the Sound File

You need to add a notification sound file named `notification.mp3` to this directory.

You can:
1. Use a free sound from websites like freesound.org or zapsplat.com
2. Create your own notification sound
3. Use a simple beep or chime sound

The sound should be:
- Short (0.5-2 seconds)
- Pleasant and not jarring
- Clear and audible
- Small file size (< 100KB recommended)

## Example Sources

- https://freesound.org/
- https://zapsplat.com/
